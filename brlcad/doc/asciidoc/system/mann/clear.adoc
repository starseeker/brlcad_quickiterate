== NAME
clear - Clears the mged command window.

== SYNOPSIS

[source]
----
clear
----


== DESCRIPTION

Clear the _mged_ command window.


== EXAMPLES

Clear the _mged_ command window.

.Enter clear to clear the mged command window.
[example]
====

mged>*clear*::
The _mged_ command window is clear.



====
