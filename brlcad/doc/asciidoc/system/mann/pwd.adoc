== NAME
pwd - Prints the current working directory.

== SYNOPSIS

[source]
----
pwd
----


== DESCRIPTION

Prints the current working directoy. See also the *cd* command.


== EXAMPLES

Print the current working directory.

.Type pwd to display the current working directory.
[example]
====

mged>*pwd*::
Returns the current working directory.
