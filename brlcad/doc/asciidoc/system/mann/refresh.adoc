== NAME
refresh - Updates the MGED display.

== SYNOPSIS

[source]
----
refresh
----


== DESCRIPTION

Updates the MGED display.


== EXAMPLES

The example shows the use of the *refresh* command to update the MGED display.

.Update the MGED display.
[example]
====
mged>*refresh*

Updates the MGED display.

====
