== NAME
mv - Changes the name of old_name to new_name.

== SYNOPSIS

[source]
----
mv {<old_name new_name>}
----


[[description]]
== DESCRIPTION

Changes the name of _old_name_ to _new_name_. Note that this does not change any references to _old_name_ that may appear in other combinations in the database. The *mvall* command will change an objects name everywhere.


[[examples]]
== EXAMPLES

The example shows the use of the *mv* command to change an object's _old_name_ to a _new_name_.

.Change an object's name.
[example]
====
mged>*mv shapea shapeb*

Changes the name of _shapea_ to _shapeb_.

====
