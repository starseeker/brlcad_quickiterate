== NAME
viewsize - Sets the size of the current viewing cube to the specified view_size (in local units).

== SYNOPSIS

[source]
----
viewsize {<view_size>}
----


== DESCRIPTION

Sets the size of the current viewing cube to the specified view_size (in local units). This size is the length of any side of the square mged display. This command is a synonym for the *size* command. See also the *view* command.


== EXAMPLES

The example shows the use of the *viewsize* command to set the mged display to a specified number of units.

.Set the size of the mged display.
[example]
====
mged>*viewsize 250*

Sets the mged display to be 250 units across.

====
