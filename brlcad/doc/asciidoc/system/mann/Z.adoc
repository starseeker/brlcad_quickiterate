== NAME
Z - Clears the mged display.

== SYNOPSIS

[source]
----
Z
----


== DESCRIPTION

Zap (i.e., clear) the _mged_ display.


== EXAMPLES

Clear the _mged_ display.

.Enter Z to clear the mged display.
[example]
====

mged>*Z*::
The _mged_ display is clear.



====
