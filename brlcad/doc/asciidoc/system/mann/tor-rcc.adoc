== NAME
tor-rcc - Creates an RCC shape that fills in the hole of a specified TOR.

== SYNOPSIS

[source]
----
tor-rcc {<torname>} {<newname>}
----


== DESCRIPTION

Creates an RCC shape that fills in the hole of a specified TOR.


== EXAMPLES

The example shows the use of the *tor-rcc* command to create an RCC to fill in the hole of a TOR.

.Create an RCC shape to fill in the hole of a TOR.
[example]
====
mged>*tor-rcc tor.s rcc.s*

Creates an RCC named _rcc.s_ to fill in the hole in the middle of _tor.s_.

====
