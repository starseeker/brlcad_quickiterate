== NAME
fb-fb - copy an image on one frame buffer to another

== SYNOPSIS

[source]
----
fb-fb [-v] [-F<output_framebuffer>] \
     [-v] <input_framebuffer> [<output_framebuffer>]
----


== DESCRIPTION

*fb-fb* copies an image from the input frame buffer onto the output frame buffer. The framebuffer may also be explicitly specified by using the *-F* flag.

The *-v* flag sets verbose mode.
