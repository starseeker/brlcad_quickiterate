== NAME
ap-pix - convert Applicon files to pix files

== SYNOPSIS

[source]
----
ap-pix [-v] <file.ap> > <file.pix>
----


== DESCRIPTION

*ap-pix* converts an Applicon drum plotter image file into a *pix*(5) file.

The Applicon file is assumed to consist of a number lines each 3456 bits long; the resulting _pix_ file thus has scanlines of 3456 pixels.
