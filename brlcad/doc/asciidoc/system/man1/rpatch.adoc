== NAME
rpatch - convert FASTGEN ASCII databases into a format suitable for

== SYNOPSIS

[source]
----
rpatch [-D] [-3] < <FASTGEN_FILE> > <OUTPUT_FILE>
----


== DESCRIPTION

This program reformats FASTGEN files to a form usable by *patch-g* . The *-D* option handles type 3 components as donuts rather than triangles. The *-3* option reads FASTGEN3 format files (coordinates are each 10 columns wide).


== EXAMPLE

rpatch < _fastgen_file_ > _file.rpatch_
