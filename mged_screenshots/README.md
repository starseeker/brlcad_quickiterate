# MGED Editing Screenshot Test Suite

This directory contains tools and screenshots for verifying MGED primitive editing
operations after refactoring.

## Scripts

### `mged_edit_test.sh` (repository root)
Generates **during-edit** and **after-edit** screenshots for all MGED-editable primitives
and all parameter-edit operations.

```
Usage: ./mged_edit_test.sh <mged_binary> <output_dir> [prefix]
```

- **during-edit**: captured after selecting the parameter-edit sub-mode from the menu —
  shows the faceplate GUI with illuminated wireframe, vertex/edge/face labels, edit menu
  on the left, and the keypoint in the status bar.  For ARB8 this captures the edge or
  face label that is most diagnostic for verifying correctness.
- **after-edit**: captured after applying the `p` (parameter) command — shows the
  modified geometry at the same view.

**Key implementation details:**
- A `screengrab` is always issued immediately after `sed <prim>` and before any `M`
  (mouse) command.  This triggers `mmenu_display()` which sets `ms_top = 1832`; without
  this initial refresh the menu hit-test fails with "mouse press outside valid menu".
- Menu item Y coordinates: `menu_y(N) = 1780 - 104*N` (N=0 title, N=1 first real item)
- All `p` parameter values are in mm matching the ~500mm default shapes from `make`.

View angle: `ae 35 25` (isometric) for clearer visibility of 3D shape changes.

Covers:
- Simple primitives (SPH, ELL, TOR, TGC, RPC, RHC, EPA, EHY, ETO, HYP, SUPERELL, PART, GRIP, HALF, EXTRUDE)
- ARB8 — all 12 Move Edges, 6 Move Faces, 6 Rotate Faces (with hierarchical menu navigation)
- BOT — Pick/Move Vertex, Edge, Triangle
- NMG — Pick/Move Edge and Vertex
- ARS — Pick/Move Vertex, Move Curve
- PIPE — Select Point, Set OD/ID/Bend, Scale OD
- METABALL — Select Point, Move Point, Scale fldstr, Set Threshold
- EBM — Set Depth

### `mged_compare.sh` (repository root)
Runs `mged_edit_test.sh` for both updated and vanilla MGED builds, then generates
side-by-side comparison images and an HTML gallery.

```
Usage: ./mged_compare.sh [updated_mged] [vanilla_mged] [out_dir]
```

Defaults:
- `updated_mged`: `brlcad_build/bin/mged`
- `vanilla_mged`: `brlcad_vanilla_build/bin/mged`
- `out_dir`: `mged_screenshots/sys`

## `sys/` — Generated screenshots

Systematically generated during+after edit pairs for every MGED-editable primitive.
Generated by `mged_edit_test.sh` with prefix `upd`.

- `upd_<prim>_<op>_during.png` — immediately after menu item selected (faceplate menu + vertex labels)
- `upd_<prim>_<op>_after.png` — after `p <value>` applied (modified geometry)

**73 during+after pairs covering:**
- SPH, ELL (3 params each), TOR (R1, R2), TGC (H, A, B, C, AB), RPC (H, r, B),
  RHC (H, r, c), EPA (H, A, B), EHY (H, A, B, c), ETO (r, D, C), HYP (H, A, B, c),
  SUPERELL (A, B, C, ABC), PART (H, V, R2), GRIP (sscale)
- ARB8: 12 Move Edges, 6 Move Faces, 6 Rotate Faces (hierarchical 2-level menu navigation)
- BOT: Pick Vertex+Move, Pick Edge+Move, Pick Triangle+Move
- NMG: Pick Edge+Move
- ARS: Pick Vertex+Move, sscale
- PIPE: Set Pipe OD (global, no prior select needed)
- METABALL: Set Threshold (global)

**Skipped primitives** (geometry not visible in scripted -c mode):
- HALF: halfspace wireframe has unbounded extent; no visual change from sscale
- EXTRUDE: `make extrude1 extrude` creates placeholder without sketch reference
- EBM: `make ebm1 ebm` creates placeholder without `.bw` bitmap path
- NMG sscale: default NMG bounding box is 0.0005mm (invisible at default zoom)
