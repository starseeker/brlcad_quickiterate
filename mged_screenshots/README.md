# MGED Editing Screenshot Test Suite

This directory contains tools and screenshots for verifying MGED primitive editing
operations after refactoring.

## Scripts

### `mged_edit_test.sh` (repository root)
Generates **during-edit** and **after-edit** screenshots for all MGED-editable primitives
and all parameter-edit operations.

```
Usage: ./mged_edit_test.sh <mged_binary> <output_dir> [prefix]
```

- **during-edit**: captured immediately after entering a solid-edit sub-mode — shows the
  faceplate GUI with illuminated wireframe, parameter labels, and edit menus on the left
- **after-edit**: captured after applying the edit parameter command

View angle: `ae 35 25` (isometric) for clearer visibility of 3D shape changes.

Covers:
- Simple primitives (SPH, ELL, TOR, TGC, RPC, RHC, EPA, EHY, ETO, HYP, SUPERELL, PART, GRIP, HALF, EXTRUDE)
- ARB8 — all 12 Move Edges, 6 Move Faces, 6 Rotate Faces (with hierarchical menu navigation)
- BOT — Pick/Move Vertex, Edge, Triangle
- NMG — Pick/Move Edge and Vertex
- ARS — Pick/Move Vertex, Move Curve
- PIPE — Select Point, Set OD/ID/Bend, Scale OD
- METABALL — Select Point, Move Point, Scale fldstr, Set Threshold
- EBM — Set Depth

### `mged_compare.sh` (repository root)
Runs `mged_edit_test.sh` for both updated and vanilla MGED builds, then generates
side-by-side comparison images and an HTML gallery.

```
Usage: ./mged_compare.sh [updated_mged] [vanilla_mged] [out_dir]
```

Defaults:
- `updated_mged`: `brlcad_build/bin/mged`
- `vanilla_mged`: `brlcad_vanilla_build/bin/mged`
- `out_dir`: `mged_screenshots/sys`

## Directories

- `sys/` — Systematically generated screenshots from `mged_edit_test.sh`
  - `upd_<prim>_<op>_during.png` — during-edit state (updated MGED)
  - `upd_<prim>_<op>_after.png` — after-edit state (updated MGED)
  - `van_<prim>_<op>_*.png` — same shots from vanilla MGED (generated by compare script)
  - `compare/compare_<prim>_<op>_*.png` — side-by-side comparisons
  - `gallery.html` — HTML gallery viewer
