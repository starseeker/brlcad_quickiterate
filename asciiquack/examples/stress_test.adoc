= PDF Layout Stress Test
Stress Tester <test@example.com>
v1.0, 2026-03-07: Initial stress-test release
:pdf-page-size: Letter

This document deliberately exercises every layout scenario that has
historically caused overlap, clipping, or mis-sizing in the PDF back-end.
Each section targets a specific risk area so that a visual or automated
diff will immediately reveal regressions.

== 3. Admonitions

== 17. A4 Page Size
// ─────────────────────────────────────────────────────────────────────────────

This section heading and its content are rendered at the same settings as
the rest of the document.  To test A4 layout specifically, generate the PDF
with `-a pdf-page-size=A4`.  All margin and content-width calculations scale
with the page size; the A4 content area is slightly narrower and taller than
Letter.

----
$ asciiquack -b pdf -a pdf-page-size=A4 examples/stress_test.adoc
----

The code block above tests that the layout calculation on an A4 page is
consistent with the Letter layout demonstrated throughout this document.
